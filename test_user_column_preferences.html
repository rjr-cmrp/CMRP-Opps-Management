🧪 User-Specific Column Preferences Test Suite

📋 Test Overview

This test suite validates the user-specific column visibility preferences implementation for the CMRP Opportunities Management application.

✅ What We're Testing:

  • User Authentication: Login with different users
  • Column Preferences Storage: Save user-specific column visibility settings
  • Column Preferences Loading: Load user-specific settings on login
  • Database Persistence: Verify preferences persist across sessions
  • User Isolation: Ensure different users have independent preferences
  • Migration: Test localStorage to database migration
